# Competency Library v2 (Seed)
이 문서는 시스템 시작 시 기본 제공되는 10개의 핵심 역량 초기값(Seed Data)을 정의한다.
각 역량은 평가 기준(체크리스트 5개 항목)과, 체크된 수에 기반한 1~5 레벨의 앵커(기준) 문장으로 구성.

## 공통 레벨 산출 규칙
- 선택된 체크리스트 항목 수 = N (0~5)
- Level = N (N=0일 경우 L1로 간주 혹은 별도 표기 없음. 보통 최소 1개 체크 시 L1, 5개 체크 시 L5.)

---

## 1. 문제 해결 (Problem Solving)
- **정의**: 복잡한 문제를 분석하고, 여러 대안을 비교해 가장 합리적인 해결책을 도출하여 실행한다.
- **체크리스트**
  - [ ] 문제의 근본 원인을 표면적인 증상과 분리해 명확히 정의함
  - [ ] 2가지 이상의 해결 대안을 도출하고 각각의 장단점을 비교함
  - [ ] 한정된 자원 및 현실적 제약을 고려해 최적안을 선택함
  - [ ] 해결책을 실제로 실행에 옮김
  - [ ] 실행 후 발생한 예상치 못한 부작용을 극복하거나 결과를 회고해 개선점을 도출함
- **레벨 앵커**
  - L1: 당면한 문제의 현상을 인지하고, 문제를 풀어보려는 기본적인 시도를 함.
  - L2: 문제 상황을 파악하고 대안을 고안해 구체적인 실행에 옮김.
  - L3: 문제의 근본 원인을 고민하고 가장 타당한 해법을 찾아 완료함.
  - L4: 다양한 시나리오와 제약을 염두하고 해결 과정 내 문제를 관리함.
  - L5: 고차원적인 분석과 실행으로 문제 해결 후 구조적 개선까지 이끌어냄.

## 2~10. 코어 역량 (안)
(실제 구현 및 Seed 데이터에서는 다음과 같은 테마의 역량이 확장 제공됨)
2. **커뮤니케이션 (Communication)**: 맥락 공유, 적극적 경청, 명확한 의견 제시
3. **실행력 (Execution)**: 일정 관리, 책임감, 끈기, 목표 달성
4. **협업 & 팀워크 (Collaboration)**: 이견 조율, 동료 지원, 책임 분담
5. **학습 민첩성 (Learning Agility)**: 빠른 적응, 피드백 수용, 새로운 도구/방식 습득
6. **고객/유저 중심 (Customer Focus)**: 피드백 반영, 니즈 파악, 유저 관점 도출
7. **기획 및 분석 (Planning & Analysis)**: 데이터 기반 의사결정, 로드맵 수립
8. **리더십 및 영향력 (Leadership)**: 구성원 동기 부여, 비전 제시
9. **창의성 및 혁신 (Creativity/Innovation)**: 기존 방식 타파, 새로운 아이디어 실현
10. **적응력/회복탄력성 (Resilience)**: 실패 대응, 상황 변화에 대한 유연한 태도
